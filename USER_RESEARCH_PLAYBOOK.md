# USER RESEARCH PLAYBOOK — Dawayir

## Scope
- Industry: Mental wellness + reflective AI
- Approach: Behavior-first (opinions are secondary to observed behavior)
- Core rule: ask "حدثني عن آخر مرة..." not "هل تفضل...؟"

## Part 1 — Interview Script (5 Questions)
Use these exactly as open-ended prompts:

1. حدثني عن آخر مرة حسّيت إن دماغك متكدسة ومش عارف تبدأ منين.  
2. حدثني عن آخر مرة استخدمت حل عشان تفك الزحمة الذهنية (شخص/ورقة/تطبيق). إيه اشتغل وإيه فشل؟  
3. حدثني عن آخر قرار مهم اتأخر منك. إمتى عرفت تتحرك؟  
4. حدثني عن آخر مرة خرجت من جلسة/محادثة وحسّيت إنك كسبت اليوم. إيه اللي اتغيّر بعدها؟  
5. حدثني عن آخر مرة بدأت حل لتحسين حالتك وبعدين وقفته. ليه وقفته تحديدًا؟

### Interview Flow (30-35 min)
1. Warm-up (3 min): تعريف سريع + تأكيد الخصوصية.
2. Context (5 min): آخر 7 أيام ومتى ظهر الضغط.
3. Core behavior (15 min): الأسئلة الخمسة بالأعلى.
4. Evidence probe (7 min): التوقيت، الأدوات، النتيجة الفعلية.
5. Wrap (3-5 min): "لو في ميزة واحدة تنقذك الأسبوع ده، تكون إيه؟"

## Part 2 — User Journey Map (6 Stages)
### 1) Trigger
- Think: "أنا مش قادر أكمل بنفس الفوضى."
- Feel: ضغط/تشوش.
- Do: تأجيل/فضفضة/تشتت.
- Pain: لا يعرف أول خطوة.
- Golden opportunity: دخول فوري بلا تعقيد.

### 2) Awareness
- Think: "هل المشكلة نفسية ولا تنظيمية؟"
- Feel: حيرة.
- Do: بحث سريع/سؤال دائرة قريبة.
- Pain: محتوى عام.
- Golden opportunity: عكس بصري لحظي للحالة.

### 3) Consideration
- Think: "أجرب أي حل؟"
- Feel: مقاومة.
- Do: مقارنة سطحية ثم تأجيل.
- Pain: الحلول بطيئة أو وعظية.
- Golden opportunity: وعد واضح "قرار واحد خلال 10 دقائق".

### 4) Decision
- Think: "أبدأ الآن ولا بعدين؟"
- Feel: تردد.
- Do: يبدأ فقط لو البداية سهلة.
- Pain: أول دقيقة غير مقنعة.
- Golden opportunity: قيمة ملموسة في أول 30 ثانية.

### 5) Use
- Think: "هل فاهمني فعلاً؟"
- Feel: احتواء أو نفور.
- Do: يتكلم/يوقف/يعيد.
- Pain: ردود عامة.
- Golden opportunity: Truth Contract واضح وقابل للتنفيذ.

### 6) Outcome
- Think: "هل في فرق بعد الجلسة؟"
- Feel: وضوح أو إحباط.
- Do: ينفذ خطوة أو لا.
- Pain: عدم المتابعة.
- Golden opportunity: تتبع تنفيذ + تذكير + قياس رجوع.

## Part 3 — Full JTBD

- Emotional Job: الشعور بالوضوح والسيطرة بدل التشتت.
- Social Job: الظهور كشخص متزن وفعّال.

## Part 4 — Top 5 Risky Assumptions
### A1
- Assumption: المستخدم يريد قرار تنفيذي أسرع من جلسة طويلة.
- Fast check: A/B (short vs long session).
- Signal: 24h action completion rate.
- If false: تقسيم التجربة مرحلتين (تفريغ + تنفيذ).

### A2
- Assumption: الدوائر تزيد الفهم ولا تزيد التشويش.
- Fast check: usability test (15 min).
- Signal: قدرة المستخدم على تفسير حالته بعد الجلسة.
- If false: تبسيط بصري وتقليل عناصر.

### A3
- Assumption: Truth Contract يرفع D1/D7 retention.
- Fast check: cohort with/without contract.
- Signal: return rate + completion rate.
- If false: تعديل صياغة/توقيت العقد.

### A4
- Assumption: الإدخال الصوتي مريح وقت الضغط.
- Fast check: track voice vs text fallback.
- Signal: voice session completion %.
- If false: مسار نصي سريع موازٍ.

### A5
- Assumption: التخصيص السياقي أهم من النصائح العامة.
- Fast check: generic vs context-aware responses.
- Signal: satisfaction + execution behavior.
- If false: تحسين طبقة السياق والذاكرة القصيرة.

## Part 5 — Survey (10 Questions)
Each question includes type + decision usage.

1. آخر 7 أيام: كم مرة حصل تشوش منعك تتحرك؟  
- Type: Scale (0-7+)
- Use: حجم الألم وتكراره

2. أول رد فعل عند التشوش؟  
- Type: MC
- Choices: أؤجل / أتكلم مع شخص / أكتب / أتشتت / غيره
- Use: تصميم نقطة الدخول

3. الحل الحالي يساعدك تاخد خطوة واضحة.  
- Type: Scale (1-5)
- Use: baseline ضد المنتج

4. أكثر شيء تكرهه في الحل الحالي؟  
- Type: MC
- Choices: عام / بطيء / غير عملي / مرهق / خصوصية
- Use: أولويات المنتج

5. أثق أن أداة رقمية تفهم سياقي الحقيقي.  
- Type: Scale (1-5)
- Use: قياس حاجز الثقة

6. النتيجة المثالية: خطوة واحدة قابلة للتنفيذ الآن.  
- Type: Scale (1-5)
- Use: التحقق من JTBD الأساسي

7. الوسيط المفضل وقت الضغط؟  
- Type: MC
- Choices: صوت / نص / مختلط
- Use: أولوية roadmap

8. لو أخذت خطوة واضحة خلال 10 دقائق، سأستخدم الأداة أسبوعيًا.  
- Type: Scale (1-5)
- Use: مؤشر تبني مبكر

9. أحتاج متابعة بعد الجلسة (تذكير/تتبع).  
- Type: Scale (1-5)
- Use: قيمة ما بعد الجلسة

10. ما الجملة التي لو سمعتها بعد الجلسة ستقول "ده اللي محتاجه"؟  
- Type: Open
- Use: تحسين copy والرسائل
