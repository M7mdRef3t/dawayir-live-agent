# Dawayir 4-Minute Video Script

## Goal

Show Dawayir as a `Cognitive OS`, not a chatbot.

The key rule for this video:

> show the system acting before you over-explain it

## Runtime

Target: `3:30 - 4:00`

## 0:00 - 0:15

### On Screen

- cinematic home screen
- logo
- `Run Live Demo`

### Voiceover

Most AI tools only answer you.

Dawayir does something different.

It renders the mind changing in real time.

## 0:15 - 0:40

### On Screen

- click `Run Live Demo`
- Gemini Live connects
- circles appear in the live session

### Voiceover

Dawayir is a live cognitive operating system.

It listens through voice, optionally sees through vision, and reshapes a live cognitive canvas during the conversation.

## 0:40 - 1:10

### On Screen

- start the live interaction
- circles move while the agent responds
- show transcript and live state

### Voiceover

Instead of waiting until the end of a chat, the agent changes the environment while the session is happening.

The three circles represent Awareness, Knowledge, and Truth.

They act as a living map of the user’s internal state.

## 1:10 - 1:35

### On Screen

- keep the live session visible
- show `Why Now`

### Voiceover

This is not a decorative animation.

Every important change can be explained through a reasoning layer we call `Why Now`.

The system shows not only what changed, but why it changed.

## 1:35 - 1:55

### On Screen

- interrupt the agent once during speech
- show that the agent stops quickly

### Voiceover

The interaction is interruption-first.

If the user cuts in, the agent stops and gives control back immediately.

That makes the session feel live, responsive, and safe.

## 1:55 - 2:20

### On Screen

- trigger optional camera or show vision-based screenshot
- show multimodal update

### Voiceover

Dawayir also supports multimodal input.

With optional vision, the system can take camera context into account and adapt the live experience accordingly.

## 2:20 - 2:50

### On Screen

- end or save the session
- open Memory Bank
- open Replay

### Voiceover

Most voice demos disappear the moment the conversation ends.

Dawayir persists each session as a replayable cognitive artifact.

Here, we can replay the session step by step and inspect the journey.

## 2:50 - 3:15

### On Screen

- show Highlight Reel
- show Signature Moment

### Voiceover

The system also extracts the most meaningful moments automatically.

That gives us a Highlight Reel and a Signature Moment that summarize what mattered most in the session.

## 3:15 - 3:35

### On Screen

- show Cross-Session Diff

### Voiceover

And because this is a system, not a one-off conversation, we can compare sessions over time.

Cross-Session Diff shows how clarity, equilibrium, overload, and the circles themselves changed across sessions.

## 3:35 - 3:55

### On Screen

- show Judge Mode
- show Fullscreen view

### Voiceover

For evaluation and presentation, we built a judge-ready layer on top of the product:

Judge Mode, Highlight Reel, Replay, and a guided demo route.

## 3:55 - 4:00

### On Screen

- back to strongest live frame or replay frame

### Voiceover

Dawayir is not a chatbot.

It is a live cognitive operating system for thinking with AI.
